# Documentation site

This is the [Docusaurus](https://docusaurus.io/) site for **Coding Agent From Scratch**.
The prose lives in `docs/`; everything else is standard Docusaurus scaffolding.

## Local development

```bash
cd website
npm install
npm run start      # http://localhost:3000 with hot reload
```

## Build a static site

```bash
npm run build      # outputs to website/build/
npm run serve      # preview the production build
```

## How the navigation works

The sidebar is **autogenerated** from the folder structure (see `sidebars.js`):

- Each subfolder of `docs/` becomes a category via its `_category_.json`.
- Page order inside a category comes from the `sidebar_position` frontmatter.

To add a page, drop a new `.md` file in the right folder with frontmatter:

```markdown
---
sidebar_position: 3
title: My New Page
description: One line for search engines and social cards.
---
```

No edits to `sidebars.js` are required.

## Structure

| Path | What it holds |
|------|---------------|
| `docs/` | All documentation content (Markdown/MDX) |
| `src/pages/index.js` | The landing page |
| `src/css/custom.css` | Theme variables and custom styles |
| `docusaurus.config.js` | Site config: navbar, footer, presets |
| `sidebars.js` | Autogenerated sidebar wiring |
